# News App - NewllyWood
A dynamic and responsive news website built using HTML, CSS, JavaScript, and NewsAPI. The app allows users to view the latest news articles on various topics and categories such as Finance, IPL, Politics, and more. Users can search for news by query, and articles are fetched in real-time from the NewsAPI.

## Features
### Dynamic News Fetching: 
Fetches news articles in real-time from NewsAPI based on default or user-defined queries.
### Category-Based Navigation: 
Allows users to browse news by predefined categories.
### Search Functionality: 
Enables users to search for specific news topics by entering a query.
### Responsive Design: 
Ensures the news app works seamlessly across different devices and screen sizes.
### Live Updates: 
Articles are fetched with live updates to ensure the latest news is displayed.
### External Links: 
Each news article includes a link to the full source for further reading.
